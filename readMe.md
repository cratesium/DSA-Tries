# 🌲 Trie (Prefix Tree) - Java Implementation

A **Trie (Prefix Tree)** is an efficient data structure used for storing and searching strings, especially useful for **prefix-based operations**.

This repository contains a clean and optimized implementation of Trie in Java along with common operations.

---

## 📌 What is a Trie?

A Trie is a tree-like data structure where:
- Each node represents a character
- Words are formed by traversing from root to leaf
- Common prefixes are shared → saves space & time

---

## ⚙️ Features

- Insert a word
- Search a word
- Check prefix existence (`startsWith`)
- Efficient prefix-based lookup
- Scalable for large dictionaries

---

## 🧠 Time Complexity

| Operation     | Time Complexity |
|--------------|----------------|
| Insert       | O(N)           |
| Search       | O(N)           |
| StartsWith   | O(N)           |

> N = length of the word
